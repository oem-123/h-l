BRIEF BIOGRAPHY

Before starting to write any code we should first attain to understand the problem and then furthermore try and break this problem down into sub-problems. I see programming as understanding a problem well enough to be able to explain it as simple as possible to a machine and being able to be easily understood by anyone coming back to it. And in this sense I am an artist, VS Code is my canvas, JavaScript is my paint, and each line of code I write is a testament to my artistic integrity.
